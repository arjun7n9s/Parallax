# PARALLAX Documentation

This directory contains analyst, admin, security, architecture, API, and operational
documentation.

## Structure

- `user/` - Analyst onboarding, verdicts, report guide, and SOC FAQ
- `admin/` - Installation, configuration, scaling, and monitoring
- `architecture/` - System design documents
- `api/` - REST API reference
- `runbooks/` - Operational runbooks for deployment and troubleshooting
- `security.md` - Bank audit security overview

## Build

```bash
mkdocs build --strict
```
